LeapQuartzComposer
==================
Author: Chris Birch
Date: 05/01/13

Quartz Composer Plugin to enable QC compositions to receive data from Leap Motion device using Leap SDK 7.

Background:

Plugin inspired by the pre SDK 7 "LeapQC" project on GitHub (see credits).
